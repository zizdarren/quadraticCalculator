This project was built with principle of MCV model,
where the opera package contain model components (model),
the calculatorUI package contain UI components (view),
and the main package contain the main java file for calculation (control).

In the main package, there are 3 java files.
The CalculatorMain.java, which is the kernel.
The Calculator.java, which is the controller for the U.
The ExpressionPC.java, which is Expression Parser and Calculation, 
using model components part to parse and calculate the expression enter by user.
